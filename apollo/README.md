## Apollo Example XML to Graphql JSON Response
This is a POC of converting XML API response to JSON Response using Apollo Graphql.

### Serve
#### Install Dependency
```
pnpm install

## OR

yarn install

## OR

npm install

```

#### Running the Server
```
node index.js
```
